Memory Game

This is a memory game based on icons that reflect Manchester!

Please click on the Start Button below to begin the game!

How to Play

There are 12 pairs all together in the game. The aim of the game is to match pairs of the 12 symbols relating to Manchester, by clicking on the cards. Click 'Start Game' to start playing the game. If a match is found the cards will stay flipped over when two consecutive cards are clicked, and when not a match if will flip back over almost immediately. Once all cards have been flipped over and matched, the game ends. You can click on the reset button to play the game again.
